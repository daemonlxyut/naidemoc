# naidemoc

> A Vue.js application of Interactive AI chat bot

## What is nAIdemoc?

nAIdemocはAIによるツッコミチャットボットです。
あなたの「ボケ」に応じた適切な「ツッコミ」を返します。
本アプリケーションはチャット形式で誰でも気軽に扱うことができます。
アプリケーションは[こちら](https://daemonlxyut.github.io/naidemoc/)のリンクからアクセスできます。
